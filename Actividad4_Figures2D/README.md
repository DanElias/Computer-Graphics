# 2D Figures in WebGL

## Technologies:
    - WebGL
    - Javascript
    - HTML
    - glMatrix

## Features:
    - Drawing Primitives:
        -Square: Triangle Strip
        -Triangle: Triangls
        -Circle/PacMan: Triangle_Fan

    - Transformations:
        - Translation: mat4.translate
        - Rotation:  mat4.rotate

    - Shaders
